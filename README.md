# 🛕 Thevaram Temples App (தேவார ஆலயங்கள்)


A comprehensive spiritual Android application designed to help devotees explore 274+ sacred Shiva temples mentioned in the Thevaram hymns. Built using the MIT App Inventor platform.


✨ Features

🔍 Smart Search: Filter temples by Districts and Saint-Poets (Appar, Sambandar, Sundarar).

📑 Detailed Insights: View 24 unique data points for each temple, including history, sacred water, and hymns.

📍 Live Navigation: Integrated with Google Maps via Activity Starter for precise directions.

🕉️ Divine UI: Features a Nataraja app icon, Shiva Lingam imagery, and spiritual emojis (🕉️ 🔱).

⏳ Graceful Exit: A dedicated 7-second exit screen with a divine farewell message.


🏗️ Project Structure (Simple View)

Screen 1 (Home): Loads all temple data from the CSV file.

Screen 2 (Search): Main dashboard for filtering data.

Screen_List (Results): Scrollable list of matching temples with emojis.

Screen 3 (Details): Displays 24 details, temple photos, and the Map button.

Screen 5 (Exit): A 7-second "Thank You" screen before closing.


📊 Dataset Description (Temple_data.csv)

The app uses a structured CSV file with 24 columns. Key indices include:
Index 2: District

Index 3: Temple Name

Index 21: Poet/Singer Name

Index 22 & 23: Latitude & Longitude (GPS)

Index 24: Image Filename
